# 03 — 实操路线图（12 个月）

> 写给你本人。两条线并行：系统线（AI 开发，你验收）+ 人的线（你学习实践）。
> 前提假设：可投入资金以"亏光也不影响生活"为上限；时间投入每周 3-5 小时。

## 先处理三个现实问题

### 1. 期权通道（重要，先想清楚再投入热情）

2026-05 八部门整治方案之后的现状：
- **富途/老虎**：两年清退期，存量只能卖出+转出，**期权卖方策略实质已不可行**（需要持续开新仓）
- **IBKR**：未被点名，历史上大陆居民可直开，但后续政策风险未知
- **换汇**：$50,000/年购汇额度且明文不允许用于境外证券投资，实际入金普遍经香港账户中转——这是全链条最脆弱的一环

**我的建议**：期权线今年只做两件零风险的事——① 学习（第二篇文档的第三阶段）；② IBKR 模拟盘练 Wheel。真金白银的期权卖方，等你模拟满 3 个月 + 通道政策明朗再说。**不要因为学了就急着用。**

### 2. 资金分层（先定结构再谈策略）

把可投资资产分四层，比例你自己定，但顺序不能乱：

```
第 1 层 安全垫    ：6-12 个月生活费，货基/存款。这层不满，下面全免谈。
第 2 层 核心资产  ：宽基定投（大盘指数），无脑、长期、不看盘。这是复利主力。
第 3 层 Sentinel  ：本项目的钱。起步 = 小到亏光不心疼的金额。
第 4 层 学费/实验 ：明确标记为"可亏光"的钱，模拟盘毕业后的期权试验等。
```

**残酷但重要的事实**：对大多数人，第 2 层贡献了终身财富的大头。Sentinel 的价值是现金流、纪律训练和上限可能性，不要让它挤占核心资产。

### 3. 交易所安全（数字货币线开工前必须做）

- 只用头部交易所（Binance/OKX 级别），开子账户给策略专用
- API key **只开交易权限，禁止提现权限**（系统铁律之一）
- 交易账户只放策略需要的钱；长期持仓放冷钱包或至少独立账户
- 开启所有安全项：2FA、提现白名单、反钓鱼码

## 每周节奏（稳定后）

- **每天**：看一眼 Telegram（<2 分钟，确认心跳正常）
- **每周日 30 分钟**：读 AI 周报 → 写三行日记 → 检查是否有升级/降级建议
- **每季度 2 小时**：策略法庭 + 资金四层再平衡
- **原则**：系统运行的时间越无聊越好。刺激感是成本，不是收益。

> 📎 **忙起来顾不上时**，退到 `02-financial-growth-path.md` 末尾的「最小可持续版本」——每周 2 分钟瞄一眼告警即可，偷懒不扣分，唯一铁律是"偷懒期不做重大改动"。

## 🧭 现在这个月你实际该做的（2026-07 真实进度锚点）

> 上面的 12 个月时间线是理想节奏。这里是对照系统**真实进度**的当下版本——按这个来，别被理想时间线的进度焦虑绑架。

**系统这条线（AI/强模型做，你只需点头）**：
- 系统工程层已很完整（AI 服务、否决、复盘、升级核查、安全加固都在，274 项测试全绿）。
- **卡点**：第一个策略 S1 太"安静"（3 年半仅 3 笔交易），dry-run 目前是空转的。
- **下一步**：让 AI 把第二个策略 S2 部署上并回测，再把模拟盘真正开机——这样 dry-run 才有交易可看。**全是模拟盘、零真实风险，你只需说一句"上"。**

**你这条线（你做，但都是低门槛版）**：
1. **消化一件真事**（5 分钟）：读 `02` 文档里「你的系统刚给了你第一课」那一节。不用懂技术，体会"信号太稀的回测不可信"这一句。
2. **交易所准备**（有空再做，不急）：如果还没弄——挑一个头部交易所、开子账户、API key **只开交易权限禁止提现**、开 2FA。这是碰真钱前必须做完的，但现在还没到碰真钱，不急于一时。
3. **资金四层划分想一想**（纸上 10 分钟）：Sentinel 用的钱，必须是"亏光也不影响生活"的那层。数字你自己定，但顺序不能乱（见本文档开头）。
4. **其它全部可以往后放**——书、日记、期权学习，哪天有精力再说。

**这个月的心法**：系统在模拟盘阶段，你的任务不是勤奋，是**练习什么都不做地看着它**。无聊是对的。

## 何时叫停（预先写下的退出条件）

预先承诺，到时不许讨价还价：
1. Sentinel 实盘 12 个月后总收益 < 同期 BTC 定投基准 且 回撤更深 → 降回 dry-run，重新审视策略
2. 任何一次发现自己绕过状态机手动干预且事后不敢写进日记 → 停止实盘一个月，只跑模拟
3. 交易所出现挤兑/提现异常传闻 → 24 小时内撤出该所全部资金，宁可错杀
4. 生活需要用到第 3 层的钱 → 无条件退出，投资永远让位于生活

## 最后一句

你在第一条消息里说"我也不贪心"。这句话值钱的程度超过整个系统的代码。系统的全部意义，是在你未来某个想贪心的深夜，替今天这个清醒的你把住门。
